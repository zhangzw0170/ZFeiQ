# ZFeiQ Textual TUI

Keyboard-first TUI for ZFeiQ powered by Textual.

## Run
```bash
cd ZFeiQ_Original
python3 -m zfeiq_tui.run
```

## Keymap
- Ctrl+D: Discover peers
- Ctrl+P: Phrases (command palette disabled to avoid conflict)
- Ctrl+E: Emotes
- Ctrl+F: Send file
- Ctrl+O: OCR
- Ctrl+Q: Quit
- F3..F7: Group ops (new/delete/rename/add/remove)

## Notes
- Broadcast target `所有在线 (广播)` and `[LOCAL]` always appear at the top of user list.
- All dialogs are keyboard driven (Enter / Esc). No buttons.
