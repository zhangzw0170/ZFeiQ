# ZFeiQ - GUI 版本要求

## 要求总览

Project3：在瑞芯微板子麒麟系统实现简单的局域网飞秋功能。

1. 可以建立无需服务器的聊天室,具有群聊天室的功能.
2. 搜索用户功能，可通过输入用户名、组名、IP等来查找我的好友.
3. 分组功能，给所有在线的用户群发消息及分组群发功能.
4. 支持表情包发送(可自定义表情包)、截图功能。
5. 上位机与板卡均正在部署 Qt 5.15.0 环境，用于开发 GUI 界面与截图功能，利于同时满足 CLI 与 GUI 用户的需求。
6. 文件传输：通过 IPMSG 协议与 TCP 连接实现稳定的文件传输，并借此实现表情包收发功能
7. 考虑调用 RK3566 的 NPU 实现根据文字信息自动推荐表情包，但需要评估
8. 加密通讯：若仍有时间，则考虑基于 IPMSG 协议的 XOR 加密机制实现；长期目标是集成 libsodium 加密库实现软件级安全的 Chacha20-Poly1305 算法。

## 主窗体

- 默认窗体大小：宽 500 x 高 800
- 窗体纵向分割为 2 部分 100x800（记为 A）和 400x800（记为 B）
- A 用于放置图标和说明文字
  - 靠上方的图标
    - 顶上放一个搜索框，用于搜索用户名和 IP
    - 用户
    - 组
    - 截图工具
  - 靠下方的图标
    - 表情包管理
    - 用户信息
    - 设置
- B 用于放置 A 打开的窗口
  - 用户页面
    - 左上角放头像，然后按上下放用户名和 IP 地址
    - 头像和用户名、IP地址下面放消息接受框
      - 消息接收框收到消息时显示对方用户名和 IP 地址
      - 收到文件时显示【文件】文件名.文件扩展名
    - 消息接受框下面放一个消息发送框，预留“表情”“截图”“常用语” 3 个按钮
    - 消息发送框右下角放一个“发送”按钮
