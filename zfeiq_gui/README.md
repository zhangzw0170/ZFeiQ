# ZFeiQ GUI

基于 PyQt5 (Qt 5.15.0) 的图形化客户端雏形。

## 界面布局

- 主窗体尺寸 500x800，左侧 100px 导航区，右侧内容区。
- 左侧导航区包含搜索框、常用入口（用户、组、截图）与辅助入口（表情、信息、设置）。
- 右侧内容区当前实现“用户”页面：头像、用户名/IP 信息、消息收发展示，以及“表情/截图/常用语”按钮和发送按钮。

## 运行方式

在项目根目录执行：

```pwsh
python .\main.py --gui
```

若系统未安装 PyQt5 5.15.0，请先执行：

```pwsh
pip install "PyQt5==5.15.0"
```
